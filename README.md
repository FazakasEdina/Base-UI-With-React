Getting started
---------------

- Clone the repository
```
git clone https://github.com/FazakasEdina/Base-UI-With-React.git
```
- Install dependencies
```
cd Base-UI-With-React
npm install
```
- Build and run the project
```
npm start
```
- Navigate to `http://localhost:3000`
